# Final MD, Final Website Project

### What I did?

So I started this project thinking I was going to build a normal-ish portfolio website in which I had a bit of funcitonality with clients. What I ended up building instead was a very complicated, multi website, functional system, in which I was able to move information back and forth with clients, have my own database of information and storage, and have one spot where I am able to interact with everyone. 


### How I did it?

I used quite a bit of chat GPT in the making of this final project. I had no idea what I was doing past some simple HTML but in understanding the core aspects of coding that we had worked on in the beginning of this semester I was able to work with the AI to get to where I wanted to go. 

I started out with just the core website building. The initial design was inspired by a friends album cover (Kai McGarry - Spin Out) in which there were multiple pieces of images all thrown together to create one cohesive picture that wasn't like any other studio people had seen but instead built up of ones I worked in. Jazz at Lincoln Center, The Bunker, Powerstation, and Symphony Hall. After creating a home page with ChatGPT I thought the aesthetics was the only thing left to do, until I discovered this website SupaBase, and that's where it all went downhill. 

Supabase is a website that is a "BaaS" Backend-as-a-Service platform, and I can honestly say this final was more of a "Coding in supabase with a website" then a portfolio in almost every single way. Supabase is the reason I am able to get clients to sign in to a login page, view a private page that shows differently for every single one of them, and create an admin page from which I can manage all of it. There is still a bunch of functionality left for me to take advantage of but overall it created a whole new project for me. 

### Problems I faced 

I cannot explain how many problems there was during this. I had to learn how to implement Java into HTML and then connect with Supabase in all kinds of different formats. I can't remember half of the troubleshooting I was doing as I was just moving through it all but I do know WHY everything wasn't working and HOW I fixed it. I tried to do almost no "vibe coding" I really wanted to understand what was happening to fix it in the future. I had quite a bit of issue originally with the java and html, and then getting the github html to work with SupaBase, and then finally the biggest issues was chat gpt and getting it to understand what was worng, what I was trying to do, and then communicating how to fix it.
